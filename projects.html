CourseAdmin
This project is currently under development by my team and I. The intended product is a graphical interface in which department administrators can use to update complex graduation requirements. This tool will also be able to export the graduation requirements in a YAML file that can then be used to create an interactive graduation checklist for students to keep track of their degree progress. We also plan to create a suggestion tool for students that will be able to suggest potential majors and minors based on courses already completed. Through this project, I hope to strengthen my web application skills, as well as become acquainted with database management.
